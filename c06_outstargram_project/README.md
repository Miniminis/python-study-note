## Function View vs. Class Based Bivew(CBV)

- 함수형뷰
    - if 문으로 get, post 등 매서드별로 분기처리 
    - 404 등 에러 페이지 이용가능 
- 클래스뷰
    - http method 별 처리시 매서드 명을 사용함으로써 가독성이 높아지고, 보다 깔끔해짐
    - 제너릭뷰, 믹스인 : 개발 생산성과 코드 재사용성이 높아짐
